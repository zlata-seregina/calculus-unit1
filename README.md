# calculus-unit1
# MATH 1211 – Unit 1: Function Classification and Graph Analysis

# Overview
This repository contains my Unit 1 assignment for **MATH 1211 – Calculus**. The purpose of this project was to analyze and classify different types of mathematical functions by interpreting their graphs and applying fundamental calculus concepts.
The assignment demonstrates my ability to identify function characteristics, determine domains and ranges, justify mathematical reasoning, and communicate solutions using clear explanations supported by academic references.

## Learning Objectives
The project focuses on the following concepts:
- Identifying different types of functions from their graphs
- Classifying rational, exponential, logarithmic, and trigonometric functions
- Determining the domain and range of each function
- Explaining mathematical reasoning using graphical analysis
- Applying proper academic citations using APA format

## Skills Demonstrated
- Mathematical reasoning
- Function classification
- Graph interpretation
- Domain and range analysis
- Technical writing
- Academic research and citation (APA)

## Reflection
Completing this assignment strengthened my understanding of how different functions behave and how their graphical properties reveal important mathematical characteristics. It also improved my ability to communicate mathematical concepts clearly while supporting my work with credible academic sources.
